# 鬼だくん福笑い

* ゲーム実況者「[汗だく](http://com.nicovideo.jp/community/co17194)」の生放送応援用に作成したもの

## 開発方法

* node.js のインストール
    * 動作確認環境
        * node: v7.9.0
        * npm: 4.2.0
* `npm install`
* `gulp`
    * サーバーが起動する
    * 変更があれば自動でサーバー再起動 & 静的ファイルを minify して再配置する
